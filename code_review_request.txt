I have addressed the issue where input fields (TextBox) were becoming unreadable upon focus (white text on white background) and ComboBox options were not showing correctly.

Changes:
1. Modified App.axaml to set RequestedThemeVariant="Dark". This ensures Avalonia's FluentTheme uses dark-appropriate colors for focused states, popups, and dropdowns.
2. Added global styles in App.axaml for TextBox to prevent its internal Border element from overriding the custom dark backgrounds used in the views during focus and pointerover states.
3. Added a global style for ComboBox pointerover state to maintain consistency.

The project builds successfully with these changes.
